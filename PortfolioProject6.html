GAMEPLAY

This are the images of adding after effects such as bloom and cpntrast using the post processing feature that is available in unity which makes it more realistic and immersive for the audience. My game was too resemble a medieval dungeon in hell hence i have used high contrasting lights to make the overall atmosphere to become reddish and bright almost resembling the enviroment the user is in which is the underworld

I have added himmer effects onto some of the elements across the dungeon which is to make the overall envrioment to have some contrast which allows me to be able to add mroe colour and variations to the elements so the user is able to see all of it at once.

I also made th interactive obejcts have their own effects so that the user will be able to differentiate inanimate objects and the objects that they can interact with which allows the ease of use and avoid confusion or motion sickness as wearing a VR headset for long hours can cause alot of stress and possible tiredness, hence its important for obejcts to be visible and obvious.

This is the whole gameplay loop of the game which lasted around 4 minutes as i was just demoing the game to test its functionality as well as the overall expereince. I am also able to rest the height and motion that the user is going to use such as picking up objects and throwing them for example to ensure everything is at an optimal height and playable zone.
